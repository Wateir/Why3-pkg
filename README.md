# Why3-pkg
Just a pkgbuild for why3 on Arch Linux

## Installation

```
cd why3-pkg  
[AURhelper] -S ocaml-menhir  
makepkg-si
```

### Uninstall

`sudo pacman -Rns why3-git ocaml-menhir`
